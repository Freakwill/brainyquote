# brainyquote
Get quotes from brainyquote.com (written in python)

## requirements
* requests, bs4
* pybrainyquote (created by the author)
* yaml (Optional)

## How to use it
Happy to use it.

## Individuation

Write it down into `quotes.yaml`, if you find a good quote.

Set topic what you like in `quote.py`
